# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.1] - 2026-01-10

### Changed
- Refactored plugin internals for better code consistency and robustness.
- Improved the script-loading logic to be more defensive and efficient.

## [0.1.0] - 2026-01-10

### Added
- Initial release of the Codeworks Umami plugin.
- Seamlessly integrates Umami Analytics with WordPress.
- Provides an admin settings page to configure the Umami Website ID and Script URL.
- Loads the Umami script with `async` and `defer` attributes for better performance.
